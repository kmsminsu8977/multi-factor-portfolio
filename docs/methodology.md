# 방법론 노트

## 1. 프로젝트 목적

가치, 모멘텀, 퀄리티, 저변동성 점수를 합성해 포트폴리오 후보를 선별한다.

## 2. 핵심 질문

복수 팩터를 같은 척도로 표준화하면 어떤 종목이 안정적으로 상위권에 남는가?

## 3. 입력 데이터

- 파일: `data/sample/factor_scores.csv`
- 성격: 강의자료나 외부 원천을 복제하지 않은 합성 샘플
- 목적: 계산 흐름과 결과 해석을 작은 표본으로 먼저 검증

## 4. 계산 절차

1. 팩터별 원점수를 z-score로 바꾸고 동일 가중 composite score를 계산한다.
2. 상위 종목은 composite score에 비례해 배분하되 concentration을 확인한다.
3. 종목명과 점수는 실제 종목 데이터가 아닌 합성 예시다.

## 5. 산출물 해석

`outputs/tables/baseline_results.csv`는 프로젝트별 baseline 결과입니다. 이 표는 최종 연구 결론이 아니라, 이후 실제 데이터 연결과 상세 분석을 시작하기 위한 구조 검증용 산출물입니다.

## 6. 한계

- 합성 데이터 기반이므로 통계적 유의성이나 투자 가능성을 주장하지 않습니다.
- 거래비용, 세금, 슬리피지, 유동성 제약은 일부 프로젝트에서만 단순 반영했습니다.
- 프로젝트별 정교한 모델링은 별도 후속 작업에서 확장해야 합니다.
