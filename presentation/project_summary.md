# 발표 초안

## 1. 문제 정의

복수 팩터를 같은 척도로 표준화하면 어떤 종목이 안정적으로 상위권에 남는가?

## 2. 데이터와 가정

- 합성 샘플 데이터: `data/sample/factor_scores.csv`
- 원본 강의자료/코드 직접 사용 없음
- 재현 가능한 baseline 실행을 우선 구성

## 3. 방법

가치, 모멘텀, 퀄리티, 저변동성 점수를 합성해 포트폴리오 후보를 선별한다.

## 4. 현재 산출물

- 실행 스크립트: `python -m src.run_baseline`
- 결과 표: `outputs/tables/baseline_results.csv`

## 5. 후속 작업

- 실제 데이터 연결
- 민감도 분석
- 차트/보고서 산출
- 프로젝트별 상세 검증
